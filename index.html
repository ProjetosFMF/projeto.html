1. Apresentação

Esse trabalho tem como conceito a Automação Residencial, para fins de conforto,

segurança, mas principalmente acessibilidade. Para tal, utiliza-se a plataforma de prototipagem

eletrônica composta por NodeMCU e Arduino, para controlar, acionar e monitorar portas e a

iluminação do local. Tudo isso sendo representado em uma maquete de residência com quatro

cômodos, utilizando LED (Diodo emissor de luz), micro servos motores, LDR (Resistor

dependente de luz), motor DC e Relé. Para o controle de dispositivos, um aplicativo Android

será desenvolvido, tornando possível a utilização em Smartphones com este sistema

operacional. Uma vantagem na montagem de um protótipo de automação residencial utilizando

as plataformas citadas é o baixo custo, o que torna as ideias deste trabalho viáveis.

1.2. Reunião I

No primeiro momento foi necessário entender como funcionam os microcontroladores e

também os componentes eletrônicos que vão ser utilizados durante o desenvolvimento

do protótipo.

Foi desenvolvido um programa para fazer o acionamento de um único led e depois foi

ampliado para acionar três leds, também foi montado a simulação de um sensor de presença

utilizando LDR.

4. Reunião III

5. Reunião IV

Como dito anteriormente, foi utilizado matérias de baixo custo e fácil aquisição, com a ideia

de uma casa acessível. Neste ultimo dia também houve a utilização dos componentes para o total

funcionamento do aplicativo. Tais como jampers para conexão, a placa NodeMCU, placa Arduino,

o micro servo motor e o resistor de 2.2 k.

Sobre a programação, o firmware embarcado no Arduino e NodeMCU foram desenvolvidos em

linguagem C utilizando a própria IDE do Arduino. O aplicativo foi desenvolvido em linguagem

Java utilizando a IDE Android Studio.

6. Considerações finais

Terminado o projeto, tudo funcionou completamente, a casa está totalmente automatizada e a baixo

custo. Nos mostra que a casa tecnológica para todos está cada vez mais viável para todos.
